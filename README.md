# I find bugs for a living. Then I build things that don't have any.

I'm Alex — an SDET based in Canada, currently studying Computer Security at Georgia Tech (OMSCS).

My day job is breaking software on purpose so it doesn't break in production.  
My side projects are tools I built because I needed them and they didn't exist yet.

---

## Things I've Built

**[System Profiling Pipeline](https://github.com/GoodVibes833/system-profiling-pipeline)**  
A lab environment where I intentionally crash my own server — CPU spikes, memory leaks, slow I/O — then watch it happen in real time. Built to get better at reading systems, not just testing them.

**[SDET CS Handbook](https://github.com/GoodVibes833/sdet-cs-handbook)**  
The CS fundamentals guide I wish someone had handed me when I started. Every concept starts with a real incident, not a textbook definition. Incident-first. Interview-ready.

**[LineCut Studio](https://github.com/GoodVibes833/LineCut)**  
A local tool that watches a video, cuts the silence, and wraps Korean captions on semantic breaks — not arbitrary character limits. Built because every existing tool did it wrong.

**[MapleGuide](https://github.com/GoodVibes833/MapleGuide)**  
Canadian immigration news is scattered across 13 provincial websites in bureaucratic English. I built a scraper that finds the updates, and a generator that explains them in Korean. For the community I'm part of.

**[Canadian Retirement Calculator](https://github.com/GoodVibes833/canadian-retirement-calculator)**  
CPP, OAS, RRSP, provincial tax — Canada's retirement system is genuinely complicated. This models it year by year so you can actually see the numbers.
