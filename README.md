---

## Things I've Built

**[System Profiling Pipeline](https://github.com/GoodVibes833/system-profiling-pipeline)**  
A lab where I intentionally crash my own server — CPU spikes, memory leaks, slow I/O — and watch it happen in real time. Built to get better at reading systems, not just testing them.

**[SDET CS Handbook](https://github.com/GoodVibes833/sdet-cs-handbook)**  
The CS fundamentals guide I wish someone had handed me when I started. Every concept begins with a real incident, not a textbook definition. Incident-first. Interview-ready.

**[LineCut Studio](https://github.com/GoodVibes833/LineCut)**  
A local tool that transcribes a video, cuts the silence, and wraps Korean captions on semantic breaks — not arbitrary character limits. Built because every existing tool did it wrong.

**[MapleGuide](https://github.com/GoodVibes833/MapleGuide)**  
Canadian immigration news is scattered across 13 provincial websites in bureaucratic English. This scrapes, normalizes, and explains it in Korean — for the community I'm part of.

**[Canadian Retirement Calculator](https://github.com/GoodVibes833/canadian-retirement-calculator)**  
CPP, OAS, RRSP, provincial tax — Canada's retirement system is genuinely complicated. This models it year by year so you can actually see the numbers.
